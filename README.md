## 📝 Note Format Template

Each note file should follow this structure:

```markdown
# Topic Name

**Date:** YYYY-MM-DD  
**Source:** Book/Course/Article Name  
**Tags:** #tag1 #tag2

## Summary

Brief overview of the topic.

## Key Concepts

### Concept 1
Explanation with examples.

### Concept 2
Explanation with examples.

## Code Examples

\`\`\`python
# Example code
\`\`\`

## References

- [Link 1](url)
- [Link 2](url)

## Related Notes

- [Related Topic](link-to-other-note.md)
```
